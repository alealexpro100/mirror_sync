mirror_sync
===========

## About
Collection of scripts for synchronization of local mirror.
Supported mirrors:
* Alpinelinux
* Archlinux
* Debian (+ astralinux, mx-linux, etc.)
* Fedora
* Voidlinux
* Oraclelinux
* Openwrt
* Tinycore linux
* Cygwin
